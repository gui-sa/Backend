Scroll Animations

Nesse exemplo, Javascript é utilizado para linkar o valor do scroll e realizar um repaint no browser a cada vez que voce sobe ou desce.

Prós

A experiência do usuario que possui um bom dispositivo, Javascript Ligado e uma boa internet é LINDA.

Contra

É de se esperar que a performance será pessima. Os clientes que realmente vão conseguir usufruir dessa artemanha serão poucos.

Melhorias

Estabelecer uma interface minima que utiliza pouco recurso. Precisa pensar também no comportamento do site quando esse for responsivo.
